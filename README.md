# Instruções para execução do sistema

## Para iniciar o sistema é necessário seguir os seguintes passos:

### 1 - Iniciar o Apache e o MySql com a porta correta no XAMPP.
### 2 - Verificar a coneção do banco de dados.
### 3 - No navegador de preferencia buscar por "localhost" e colocar o nome da pasta onde estão os arquivos.
### 4 - Escolher a pasta desejada.
### 5 - Manter o Apache e o MySql sempre funcionando, ao menos que você deseje desligar o sistema, pois só assim você consegue entrar.
